### 2. 핵심 정리 (내 언어로)

- API란?
    - 손님(프론트엔드)이 메뉴판(API 명세)을 보고 점원(API)에게 주문을 하면, 점원이 주방(백엔드/DB)에서 요리를 가져와 손님에게 전달하는 과정
    - 자동으로 가져와야할 때 : useEffect
        - 컴포넌트가 마운트될 때, 예약해둔 작업을 실행
        - 의존성 배열 [ ] - 컴포넌트 생애 주기에서 딱 한 번 실행
    - 사용자가 원할 때 : onClick, onSubmit 등 이벤트 속성에 연결된 함수 내부에 호출 로직 작성
- 비동기 통신
    - fetch : 서버에 데이터 요청
    - async & await
- 보안과 환경설정
    - .env 파일을 사용하여 소중한 정보를 숨긴다
        - 반드시 .gitignore 파일에 .env를 추가할 것
