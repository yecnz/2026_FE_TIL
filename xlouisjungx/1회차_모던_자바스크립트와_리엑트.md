### 2. 핵심 정리 (내 언어로)

- HTML 기초
  - <html> : HTML 문서의 전체 시작
  - <head> : 웹페이지의 설정 정보 (제목, css연결, 문자 인코딩)
  - <h1> ~ <h6> : 제목 태그
  - <p> : 글 태그
  - <ul> : 순서 없는 리스트
  - <ol> : 순서 있는 리스트
  - 입력창 : <input type="text" name="변수명>
  - <button> : 버튼 생성

- CSS 기초
  - 다양한 속성을 학습
  - background-color : 값; -> 값은 색 마다 주어진 값으로 설정
  - margin : 값; -> 값에 따라 요소 바깥 여백 조절
  - padding : 값; -> 값에 따라 내용과 테두리 사이 여백을 조절

- React
  - 웹 페이지를 컴포넌트 단위로 쪼개서 효율적으로 만드는 도구
  - npm create-react-app 프로젝트명 -> React를 통해 개발할 프로젝트 생성
  - npm start -> 프로젝트 폴더에서 입력 시, 실행

### 3. 실습 / 과제 / 결과물

- HTML && CSS 실습
  - 코드:
    <!doctype html>
    <html lang="en">
    <head>
        <meta charset="UTF-8" />
        <meta name="viewport" content="width=device-width, initial-scale=1.0" />
        <title>Document</title>
    </head>
    <body>
        <h1>Hello, World!</h1>
        <h4>Hello</h4>
        <div>안녕하세요</div>

        <nav>
            <p>1</p>
        </nav>

        <h3>ol/li</h3>
        <ol>
            <li>Item 1</li>
            <li>Item 2</li>
            <li>Item 3</li>
        </ol>

        <section>
            <form>
                <p>
                    <label for="name">이름:</label>
                    <input id="name" type="text" placeholder="이름을 입력하세요" />
                </p>
                <p>
                    <label for="message">메시지:</label>
                    <textarea
                        id="message"
                        rows="4"
                        cols="30"
                        placeholder="하고 싶은 말을 작성하세요"
                    ></textarea>
                </p>
                <button type="submit">전송</button>
            </form>
        </section>

    </body>
    </html>
